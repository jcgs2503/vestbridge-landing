How it works

Connect

Unified tool schema for any broker. get_quote, place_order, get_positions. Your agent learns one API.

Authorize

Define mandates: allowed assets, position limits, order size caps, time windows. Every order checked before it hits the broker.

Audit

Append-only log of every action. Agent-attributed, timestamped, hash-chained. Export for compliance review.

Local-first

Broker credentials and agent keys never leave your machine. No cloud dependency. Runs as a local process.

Get started in two commands

Install VestBridge, then connect it to your AI client. That's it.

1

Install

terminal 
$ curl -sSL https://vestbridge.dev/install.sh | sh
$ pip install vestbridge
$ npx vestbridge
$ brew install vestbridge
2

Connect to your AI client

claude_desktop_config.json 
{
  "mcpServers": {
    "vestbridge": {
      "command": "vestbridge",
      "args": ["--broker", "robinhood"]
    }
  }
}
$ claude mcp add vestbridge -- vestbridge --broker robinhood
// .cursor/mcp.json
{
  "mcpServers": {
    "vestbridge": {
      "command": "vestbridge",
      "args": ["--broker", "robinhood"]
    }
  }
}
# Start VestBridge as an SSE server for remote clients
$ vestbridge --broker robinhood --transport sse --port 8080

# Connect ChatGPT, Gemini, or any MCP-over-HTTP client
# to http://localhost:8080/sse

Roadmap

From broker connectivity to verifiable compliance, one phase at a time.

Phase 1: Connect

Now

MCP server with core tool schema. Paper trading adapter. Robinhood and Interactive Brokers adapters.

Phase 2: Authorize

Next

YAML-based mandate engine with pre-trade validation. Agent identity verification for safe multi-agent investing workflow.

Phase 3: Audit

Append-only JSONL log with hash chain. vest audit verify CLI. Export to JSON/CSV. Agent session tracking.

Phase 4: Autonomous Operations

Agent session lifecycle with heartbeat. Session-scoped mandates with auto-expiry. Trigger receiver for webhooks and websockets. Scheduled actions. Dead agent detection with automatic order pulling and position freeze.
